## 🧩 Featured Engineering Projects

### 🏭 [Manufacturing BOM Comparison Tool](https://github.com/Chrissicris2605/Manufacturing-Bom-Comparison-Tool)

Public demo of an engineering automation tool built with **Python** to compare structured technical files, classify differences, and generate a formatted Excel delta report.

**Highlights:**

- compares structured engineering data by generic `code`;
- classifies rows as `Added`, `Removed`, `Modified`, or `Unchanged`;
- generates a color-coded Excel report;
- includes fictional sample data and safe public documentation;
- inspired by a real workflow where automation reduced analysis time from **~5 days to 2–3 hours**.

**Tech:** Python, pandas, openpyxl, Excel reporting, desktop-oriented automation.

---

### 🖥️ [Engineering Dual-Pane Productivity Tool](https://github.com/Chrissicris2605/Engineering-Dual-Pane-Productivity-Tool)

Public demo of a dual-pane desktop productivity tool designed to connect visual technical context with structured component data.

**Highlights:**

- displays a simplified technical diagram and component table side by side;
- highlights visual elements based on table selection;
- allows fictional components to be added during the demo session;
- demonstrates workflow-oriented UI design and reduced context switching;
- inspired by a real productivity tool that reduced task execution time by approximately **50%** and enabled onboarding in approximately **1 hour**.

**Tech:** Python, PySide6, desktop UI, visual-to-data mapping, productivity tooling.
